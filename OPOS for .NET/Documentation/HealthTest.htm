Operating Guide for the Operation Checking Tool

This document applies to EPSON OPOS ADK for .NET Version 1.14.6

Introduction    Operating Procedure


Introduction

The operation checking tool is a tool for checking whether ServiceObjects are registered correctly.
Use it for checking the operation of ServiceObjects when registering them.


Back to the top


Operating Procedure   

Run CheckHealth Utility in the EPSON OPOS .NET program group.
The operation checking tool starts up, and the following screen appears.

 

Device Display tree view

Displays a list of ServiceObjects registered in SetupPOS Utility.

The registered ServiceObjects are classified by device category, shown in a tree format.
The ServiceObjects are shown with the logical device names set in SetupPOS.

 

CheckHealth button

Checks the operation of the ServiceObjects selected in the Device Display tree view.

If no ServiceObjects are selected in the tree view, this button is disabled.
The operation check of ServiceObjects is performed at the Interactive level of the CheckHealth method.

Refer to the operation guide for the relevant ServiceObjects for the operations of the CheckHealth method.

 

CheckHealthText message box

Displays text indicating the result of the operation check.

The text appears after the operation check.
The text shows the contents of the CheckHealthText property.

Refer to the operation guide for the relevant ServiceObjects for the text set in the CheckHealthText property.

 

Exception Information message area

Displays error information when the operation check fails due to causes such as ServiceObject registration errors.

Displays detailed information of exceptions that occurred during the operation check.
Use this for resolving any problems.

Refer to the operation guide for the relevant ServiceObjects for the procedure for resolving errors.
